BreakingBot

Description of my architecture!

Go back to the contents page

Custom-built for Eurecom OS Competition, BreakingBot has as a brain a LEGO® MINDSTORMS® EV3 which is connected to five elements (two motors for the wheels, an ultrasonic sensor, a gyro sensor and a third motor for the claw). All the elements are installed in order to have a robot looking like a two-wheel drive car.

Created by Sofiène Jerbi, Thibault Petitjean (@thib774) and Florian Kohler (@FlorianKohler)

OS Course, Eurecom 2k16
